## Project Objective
A personal finance tracker to manage income and expenses efficiently, visualize spending, and maintain monthly summaries.

---

## Features
- Add income and expense transactions
- Assign categories to transactions
- Monthly summary cards for quick overview
- Pie chart for expense breakdown
- Transaction list with delete option
- Data persistence using LocalStorage
- Responsive design for mobile and desktop

---

## Technologies Used
- **HTML**, **CSS**, **JavaScript**
- **Chart.js** for graphs
- **LocalStorage** for storing data
- (Optional) React.js if used for dynamic UI
